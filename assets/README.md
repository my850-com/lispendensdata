# Assets

This folder contains images and assets for the LisPendensData website.

## OG Preview Image

For social sharing (SMS/iMessage link previews), we need a 1200x630px PNG image.

Current setup:
- OG meta tags reference: https://lispendensdata.com/assets/preview.png
- Recommended size: 1200x630px (or 1200x1200 for square compatibility)
- Format: PNG or JPG

**To create the preview image:**

Use any design tool (Canva, Figma, Photoshop) with these specs:
- Background: Navy gradient (1a3a5c to 2a5298)
- Logo: Gold "L" on navy square
- Headline: "Lis Pendens Data" in white
- Subhead: "Pre-Foreclosure Filings • 21 States"
- Features: CSV downloads, 2,000+ counties, $49 starting price
- Save as: preview.png (1200x630px)

Once created, upload to: /assets/preview.png
