# LisPendensData.com

Nationwide lis pendens data for legal professionals.

## Structure

- `index.html` - Homepage
- `form/` - Lead capture form
- `css/` - Stylesheets
- `assets/` - Images and logos
- `states/` - State-specific pages

## Deployment

This site is hosted on GitHub Pages.

## Form Submissions

Form submissions go to Google Sheets via Google Apps Script.
